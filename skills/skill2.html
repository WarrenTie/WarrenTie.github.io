Robot Car

Team Project | 2 Weeks

The robot car was built using an ESP32 microcontroller and programmed with MicroPython as the IDE. It can be remotely controlled via Bluetooth using a mobile phone. The goal of this project was to design the fastest car in a short amount of time and win the grand prix competition.

LED Cube

Solo Project | 1 Week

This is an 8×8×8 LED cube created purely for aesthetic and cool visual effects. It lights up and displays patterns based on programmed code. This project sparked my interest in electronics during high school and helped me develop my soldering skills.
